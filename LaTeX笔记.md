# 一些LaTeX笔记

> 主要记录一些小疑难，如果后续有记录整体语法的需要，可另开一个文档

1. 如果并未声明日期，那么根据\\maketitle命令的结构：
- 标题（\\title）
- 作者（\\author）
- 日期（\\date或默认当前，即编译时的日期）
显然，会标注当前日期。
所以我们可以这样做：
```latex
\date{} %想要不显示日期时的做法，可以清除日期
\maketitle
\date{December 25, 2024} %想要具体日期时的做法
\maketitle
```
此外，我们还可以选择**自定义标题格式**，此处暂且按下不表。

2. **大部分**希腊字母都是按照读音来拼写的，而亦有反例，如 \\sum 。顺带一提，积分符号为 \\int 。

3. 省略号的打法是 \\ldots 。

4. 对于又有上标又有下标的，它们的顺序**不做要求**。
